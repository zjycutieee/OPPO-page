# OPPO Project Page

Static project page for:

**Omni-Perception Policy Optimization for Multimodal Emotion Reasoning**

## Preview locally

Open `index.html` directly in a browser.

## Publish with GitHub Pages

1. Create a public GitHub repository, for example `OPPO-page`.
2. Upload the contents of this folder to the repository root:
   - `index.html`
   - `style.css`
   - `.nojekyll`
   - `assets/`
3. In the repository, go to **Settings -> Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select branch `main` and folder `/root`, then save.

After GitHub finishes deployment, the site will be available at:

`https://<your-github-username>.github.io/OPPO-page/`
